フォルダを作る
mkdir portfolio
cd portfolio

ファイルを作る
touch index.php

docker-compose.yml を作る
touch docker-compose.yml

Dockerを起動する
docker compose up -d 

Dockerを止める
docker compose down

止まったか確認
docker compose ps

// PHPMailer
PHPMailerをComposerでインストール
docker run --rm -u "$(id -u):$(id -g)" -v "$PWD":/app -w /app composer:2 composer require phpmailer/phpmailer

確認
ls vendor
結果
autoload.php composer phpmailer

vendorをインストール
docker run --rm -u "$(id -u):$(id -g)" -v "$PWD":/app -w /app composer:2 composer require vlucas/phpdotenv

確認
ls vendor | grep vlucas
結果
vlucas





オブジェクト指向
composer.json
{
  "require": {
    "phpmailer/phpmailer": "^7.0"
  },
  "autoload": {
    "psr-4": {
      "App\\": "app/"
    }
  }
}


autoload を再生成
docker run --rm -u "$(id -u):$(id -g)" -v "$PWD":/app -w /app composer:2 composer dump-autoload
